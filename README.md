#### classe operados

```java
public class Operadores {

    public static void main(String[] args) {
        // Variável 'numeroUm' declarada e inicializada com valor 20
        int numeroUm = 20;
        // Variável 'numeroDois' declarada e inicializada com valor 30
        int numeroDois = 30;

        // Imprime a soma de numeroUm e numeroDois
        System.out.println(numeroUm + numeroDois);
        // Imprime a diferença de numeroUm e numeroDois
        System.out.println(numeroUm - numeroDois);
        // Imprime o produto de numeroUm e numeroDois
        System.out.println(numeroUm * numeroDois);
        // Imprime o quociente de numeroUm dividido por numeroDois
        System.out.println(numeroUm / numeroDois);
        // Imprime o resto de numeroUm dividido por numeroDois (operador módulo)
        System.out.println(numeroUm % numeroDois);
    }
}

```
